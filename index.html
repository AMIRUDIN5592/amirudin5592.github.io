Tentang Saya

Saya adalah seorang Full Stack Developer dan Supervisor HRIS dengan pengalaman lebih dari 8 tahun di bidang pengembangan aplikasi web, desktop, dan mobile. Saya menguasai berbagai teknologi modern seperti PHP, JavaScript (ReactJS, Node.js), Python, Golang, Java (Spring), serta pengembangan aplikasi mobile dengan React Native. Saya berpengalaman dalam perancangan dan integrasi API, penerapan kontainerisasi menggunakan Docker, serta pengelolaan database dengan PostgreSQL, MySQL, dan SQL Server. Di bidang ERP dan HRIS, saya terbiasa melakukan analisis kebutuhan, desain sistem, pengembangan report, hingga implementasi dan support pengguna pada platform seperti Dynamics NAVISION, AXAPTA, dan SAP HR. Saya juga memiliki ketertarikan dan kompetensi dalam pengembangan solusi berbasis data, termasuk implementasi machine learning dan AI, baik untuk otomasi bisnis maupun analisis prediktif (misalnya: HR analytics, prediksi turnover karyawan, dan sistem rekomendasi). Saat ini, saya tengah menempuh studi Magister Ilmu Komputer di Universitas Budi Luhur untuk memperkuat fondasi keilmuan, khususnya di bidang data science, AI, dan pengembangan sistem terintegrasi.
